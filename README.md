Simple **CMake** boilerplate to build your C++ projects.

## HOW it Works

**CMake** makes you make file and **make** build your C++ project.

Your execurable will be: *cmake_bp/build/src/cmake_bp*.

## Basic Use

    cd cmake_bp
    chmod a+x build.sh
    build.sh

## Partial Builds

    cd cmake_bp/build
    make help
    make
